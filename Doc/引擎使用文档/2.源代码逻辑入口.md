# main文件
路径：Source\Engine\Main\Windows\main.cpp
wWinMain：逻辑入口，初始传入资源hInstance、lpCmdLine
    hInstance：初始化窗口
    lpCmdLine：启动命令
WindowsPlatform.PreInit：预初始化操作
    WNDCLASS：定义一个Window类型所需的全部信息
    RegisterClassW：使用Unicode传递消息
    OleInitialize：初始化COM库，以启用部分系统应用程序功能。
    GetWindowsVersion：查看注册表获得windows系统版本
Engine.Main：引擎初始化
    WindowsPlatform.GetCurrentThreadID：获取线程id
    DateTime.Now：获取时间
    EngineService.Sort：引擎模块基于Order排序，如音频、游戏设置
        初始化全局变量services
        第一个模块初始化后才能进行Sort
    CommandLine.Parse：解析启动参数，得到CommandLine.Options
WindowsPlatform::Init
    PlatformBase.Init：初始化随机种子
    Win32Platform.Init：
        QueryPerformanceFrequency：初始化系统计时器精度(微秒)
        PlatformBase.Fatal：报错时通过StackFrame输出错误定位信息
        CpuInfo：统计CPU信息，核心数、缓存
        DeviceId：统计系统信息，计算机名、用户名、mac地址、内存、CPU算力
        WSAStartup：初始化网络
    AttachConsole：启动控制台
        通过cmd或powershell获得后台日志
            https://docs.flaxengine.com/manual/editor/advanced/command-line-access.html
            start /w E:\_Git\SekiaEngine\Binaries\Editor\Win64\Development\FlaxEditor.exe -std -project E:\_Git\SekiaEngine\Doc\SampleProjects\newProject
        NtSetTimerResolution：ntdll.dll、设置设备的计时器精度为1毫秒
